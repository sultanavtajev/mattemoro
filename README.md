MatteMoro Android App

MatteMoro er en morsom og enkel app som hjelper barn med å få teken på
addisjon.

Appen er laget for små barn, omtrent i alderen 6-10 år. Jeg valgte denne aldersgruppen fordi det er
viktig å starte tidlig med matte. Og siden mange små barn ikke kan lese ennå, har jeg prøvd å gjøre
appen så enkel og visuell som mulig.
